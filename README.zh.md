平时关注几个方向：
- **AI 驱动的敏捷开发** —— 探索怎么让 AI 参与整个软件开发生命周期，从需求到交付

- **实时量化交易系统** —— 对低延迟、高可靠的自动化交易系统感兴趣

技术栈方面，主要用 **TypeScript、React、Next.js、Node.js** 做前端和工具，**Python、Java** 做后端和量化相关的开发。
